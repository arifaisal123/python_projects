Dependencies:
pip install art
pip install Random-Word

How to Play:
python hangman.py

-> You need to guess a word of n-th character, randomly generated. 
-> On each iteration, you will guess a letter, and the blanks will be updated, if the letter is the correct one. 
-> You will win the game if you can guess the word right with remaining lives. You will lose the game if you do not have any more lives left!
