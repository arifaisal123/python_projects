YayOrNay is a text-based guessing game!

On Windows, the game can be played by clicking the executable file YayOrNay.exe, which is inside the dist folder.
Game Rules:

- The game takes a 3-digit user input for digits. 
- Based on the inputs, the game tells if the user has given correct input or not. 
- The user will get maximum 10 guesses to get towards the correct answer!


# The game has been compiled to play on Windows using pyinstaller.
